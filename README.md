# Windows Security Analyzer - Анализатор безопасности Windows

##  Суть проекта

Этот **анализатор безопасности Windows** - консольное приложение на Python, которое выполняет базовые проверки безопасности операционной системы. Программа сканирует ключевые аспекты системы и предоставляет отчет о потенциальных уязвимостях.

## Основные возможности

### 1. **Информация о системе**

- Определение версии Windows
    
- Проверка прав администратора
    
- Информация о процессоре и архитектуре
    
- Имя компьютера и пользователя
    

### 2. **Проверки безопасности**

- **Брандмауэр Windows** - статус профилей (частный, публичный, доменный)
    
- **Обновления системы** - последние установленные обновления
    
- **Антивирусная защита** - статус Защитника Windows и сторонних антивирусов
    
- **Сетевая безопасность** - активные подключения и открытые порты
    
- **Учетные записи** - список пользователей и активность учетки администратора
    
- **Общие ресурсы** - расшаренные папки и доступы
    

##  Как пользоваться программой

### **Установка и запуск:**

1. Сохраните код в файл `security_analyzer.py`
    
2. Убедитесь, что у вас установлен Python IDLE
    
3. Запустите программу:
    
    bash
    
    python security_analyzer.py
    

### **Рекомендуемый способ запуска:**

bash

# Для получения полного доступа запустите от имени администратора:
python security_analyzer.py
# ИЛИ щелкните правой кнопкой -> "Запуск от имени администратора"

### **Главное меню:**
1.  Запустить полную проверку безопасности
2.  Проверить только брандмауэр
3.  Проверить только сетевую безопасность
4.  Проверить антивирусную защиту
5.  Показать информацию о системе
6.  Выход

##  Как работает код

### **Ключевые классы и функции:**

1. **`SecurityAnalyzer`** - основной класс анализатора
    
    - `get_system_info()` - сбор информации о системе
        
    - `check_windows_firewall()` - проверка брандмауэра
        
    - `check_windows_updates()` - проверка обновлений
        
    - `check_antivirus_status()` - проверка антивируса
        
    - `show_security_summary()` - вывод итогового отчета
        
2. **`check_admin_rights_correct()`** - корректная проверка прав администратора
    
    - Использует несколько методов проверки
        
    - Учитывает особенности Windows
        
3. **`run_command()`** - безопасное выполнение команд
    
    - Обработка таймаутов
        
    - Корректная кодировка для русского языка (cp866)
        

### **Особенности реализации:**

- **Кодировка cp866** - для корректного отображения русского текста в консоли Windows
    
- **Обработка ошибок** - программа продолжает работу при ошибках в отдельных проверках
    
- **Альтернативные методы** - если один способ проверки не работает, используется другой
    
- **Таймауты** - предотвращение зависания при выполнении команд
    

##  Что проверяет программа

### **Критические параметры:**

-  Включен ли брандмауэр
    
-  Работает ли антивирусная защита
    
-  Открыты ли опасные порты (21, 22, 23, 135, 445, 3389)
    
-  Активна ли учетная запись администратора
    
-  Наличие общих ресурсов
    

### **Рекомендации:**

-  Обновление системы (если прошло больше 30 дней)
    
-  Проверка общих папок
    
-  Отключение ненужных служб

##  Важные замечания

1. **Права администратора** - некоторые проверки требуют повышенных привилегий
    
2. **Антивирусы** - программа может вызывать срабатывание антивирусов
    
3. **Ограничения** - инструмент предназначен для базовой диагностики
    
4. **Безопасность** - программа только анализирует, не изменяет настройки
